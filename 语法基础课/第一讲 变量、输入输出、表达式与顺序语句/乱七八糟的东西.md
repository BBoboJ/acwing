- 变量

    ```c++
    bool false/true					    1B
    
    char 'c', 'a', ' ', '\n'			1B	%c
    
    int	 -2^31 ~ 2^31-1					4B	%d
    
    long long -2^63 ~ 2^63-1			8B	%lld
    
    float 1.23, 1.23e2, 6-7位有效数字	 4B	 %f
    
    double 15-16位有效数字				8B	%lf
    
    long double 18-19位有效数字			16B	 %llf
    ```

- 浮点数的比较（精度问题）

    ```c++
    const double eps = 1e-6;
    fabs (a-b) <= eps;		// 相等
    a>b && fabs(a-b)>1e-6;	// 大于
    a<b && fabs(a-b)>1e-6	// 小于
    ```
